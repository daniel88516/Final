# Benchmarks

Benchmark scripts and local results for the diagnostics applets live here.

Planned comparisons:

- `bbfscheck` vs `df`, `find`, and `stat`
- `bbtop` vs `ps` and `top`
- `bbnetmon` vs `ss` and `netstat`

Local benchmark output should go under `benchmarks/results/local-*` and should not be committed unless it is small and useful for the final report.
