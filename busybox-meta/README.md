# BusyBox Metadata

This directory stores the project-owned BusyBox build metadata for the diagnostics applets.

Run `make integrate` to copy these files, along with `src/applets/` and `src/libdiag/`, into `third_party/busybox/diagnostics/`.

The generated files inside the BusyBox submodule are not the source of truth. Edit files here and under `src/` instead.
